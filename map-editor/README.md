# Редактор карт

Отдельное приложение для создания и редактирования карт для игры.

## Установка

```bash
cd map-editor
npm install
```

## Запуск

```bash
npm run dev
```

Приложение запустится на `http://localhost:3001`

## Использование

### Инструменты

1. **Ландшафт** - Выбор типа поверхности (трава, земля, булыжник и т.д.)
2. **Тип тайла** - Выбор типа тайла (пол, стена, дверь, выход, портал)
3. **Декор** - Размещение декоративных объектов (деревья, камни, фонтаны и т.д.)
4. **Здание** - Размещение зданий на карте
5. **NPC** - Размещение NPC персонажей
6. **Животное** - Размещение животных
7. **Стереть** - Удаление объектов с карты

### Управление

- **ЛКМ** - Разместить/изменить тайл или объект
- **Зажать ЛКМ и двигать** - Рисовать непрерывно
- **Кнопки масштаба** - Увеличить/уменьшить карту
- **Кнопка Home** - Сбросить позицию и масштаб

### Сохранение и загрузка

- **Сохранить карту** - Экспортировать карту в JSON файл
- **Загрузить карту** - Импортировать карту из JSON файла
- **Очистить карту** - Создать новую пустую карту

### Формат карты

Карта сохраняется в формате JSON со следующей структурой:

```json
{
  "tiles": [...],      // Массив тайлов
  "buildings": [...],  // Массив зданий
  "npcs": [...],       // Массив NPC
  "animals": [...],    // Массив животных
  "startPos": {...},   // Стартовая позиция игрока
  "width": 120,        // Ширина карты
  "height": 80         // Высота карты
}
```

## Интеграция с игрой

Сохраненные карты можно использовать в основной игре, импортировав их в функцию генерации карт.
