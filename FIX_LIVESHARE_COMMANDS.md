# Исправление ошибки "command lifeshare.signout not found"

## Проблема
Ошибка означает, что Live Share установлен, но команды не активированы.

## Решение (по порядку):

### 1. Перезагрузите окно Cursor (БЫСТРОЕ РЕШЕНИЕ)

1. Нажмите `F1` или `Ctrl + Shift + P`
2. Введите: **`Developer: Reload Window`**
3. Нажмите Enter
4. Дождитесь перезагрузки окна

После перезагрузки попробуйте команду снова.

---

### 2. Если не помогло - Перезапустите Cursor полностью

1. Закройте **все** окна Cursor
2. Откройте Cursor снова
3. Откройте ваш проект
4. Попробуйте команду: `F1` → `Live Share: Start Collaboration Session`

---

### 3. Проверьте, активировано ли расширение

1. Нажмите `Ctrl + Shift + X` (панель расширений)
2. Найдите "Live Share"
3. Убедитесь, что оно **включено** (Enable)
4. Если было отключено — включите и перезагрузите окно

---

### 4. Войдите в аккаунт Live Share

Возможно, нужно сначала авторизоваться:

1. Нажмите `F1` или `Ctrl + Shift + P`
2. Введите: **`Live Share: Sign In`**
3. Выберите способ входа (GitHub или Microsoft)
4. Войдите через браузер
5. После входа попробуйте команду снова

---

### 5. Проверьте правильные команды Live Share

**Основные команды:**
- `Live Share: Start Collaboration Session` - начать сессию
- `Live Share: Join Collaboration Session` - присоединиться
- `Live Share: Sign In` - войти в аккаунт
- `Live Share: Sign Out` - выйти из аккаунта

**НЕ используйте:**
- ❌ `lifeshare.signout` (неправильное написание)
- ✅ `Live Share: Sign Out` (правильная команда)

---

### 6. Альтернативный способ - через панель статуса

1. Посмотрите в **правый нижний угол** Cursor
2. Найдите иконку **Live Share** (два кружка или "LS")
3. Кликните по ней
4. Выберите "Start Collaboration Session"

---

### 7. Если ничего не помогает - Переустановка

```powershell
# Удалить расширение
cursor --uninstall-extension ms-vsliveshare.vsliveshare

# Установить заново
cursor --install-extension .\liveshare.vsix

# Перезапустить Cursor
```

---

## Быстрая проверка после исправления:

1. ✅ Перезагрузите окно: `F1` → `Developer: Reload Window`
2. ✅ Попробуйте: `F1` → `Live Share: Start Collaboration Session`
3. ✅ Если просит авторизацию: `F1` → `Live Share: Sign In`

---

## Если Live Share все еще не работает:

Используйте **Git + GitHub** для совместной работы (уже настроено):

1. Оба разработчика клонируют проект с GitHub
2. Работаете локально
3. Делаете `git push` и `git pull` для синхронизации

Это надежный способ совместной работы! ✅
