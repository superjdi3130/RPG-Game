# Как найти иконку Live Share в Cursor

## Где должна быть иконка Live Share:

### Местоположение 1: Статус-бар (правый нижний угол)
- Иконка появляется только когда сессия **активна**
- Если сессия не запущена — иконки не будет
- Ищите два переплетенных кружка или текст "Live Share"

### Местоположение 2: Панель активности (Activity Bar)
- Левая панель Cursor
- Иконка может быть внизу списка
- Ищите иконку с двумя кружками

### Местоположение 3: Через команды (самый надежный способ)

**Вместо поиска иконки используйте команды:**

1. **Начать сессию:**
   - `F1` или `Ctrl + Shift + P`
   - Введите: `Live Share: Start Collaboration Session`
   - Нажмите Enter

2. **Проверить статус:**
   - `F1` → `Live Share: Show Session Status`

3. **Открыть панель Live Share:**
   - `F1` → `Live Share: Focus on Chat View`

## Если иконки нет — проверьте:

### 1. Запущена ли сессия?
- Иконка появляется только когда сессия активна
- Если вы только подключились, но не создали сессию — иконки не будет

### 2. Проверка через команду:
```
F1 → Live Share: Show Session Status
```
Если команда работает — Live Share установлен и работает.

### 3. Перезапуск Cursor:
- Закройте Cursor полностью
- Откройте снова
- Попробуйте команду: `F1` → `Live Share: Start Collaboration Session`

## Альтернативный способ — через панель расширений:

1. `Ctrl + Shift + X` (открыть расширения)
2. Найдите "Live Share"
3. Кликните на него
4. В правой панели должна быть информация о расширении
5. Там может быть кнопка "Start Session" или информация о статусе

## Проверка работы без иконки:

Даже если иконки нет, Live Share может работать через команды:

1. **Запустите сессию:**
   ```
   F1 → Live Share: Start Collaboration Session
   ```

2. **Если появилась ссылка** — всё работает!

3. **Откройте чат:**
   ```
   F1 → Live Share: Focus on Chat View
   ```

4. **Проверьте статус:**
   ```
   F1 → Live Share: Show Session Status
   ```

## Если команды не работают:

1. Переустановите Live Share:
   ```bash
   code --uninstall-extension ms-vsliveshare.vsliveshare
   code --install-extension ms-vsliveshare.vsliveshare
   ```

2. Перезапустите Cursor

3. Авторизуйтесь:
   ```
   F1 → Live Share: Sign In
   ```

4. Попробуйте команду снова:
   ```
   F1 → Live Share: Start Collaboration Session
   ```
