РЕКОМЕНДАЦИИ ПО ОПТИМИЗАЦИИ:
================================================

1. УДАЛИТЬ НЕИСПОЛЬЗУЕМЫЕ ДЕКОРАЦИИ из кода:
   - decor1-17 (если не используются в игре)
   - stone1-6 (если не используются)
   - grass1-6 (если не используются)
   - box1-5 (если не используются)
   
   Это уменьшит количество попыток загрузки файлов.

2. УПРОСТИТЬ imageLoader:
   - Удалить неиспользуемые ключи из SpriteMap
   - Группировать похожие спрайты

3. LAZY LOADING:
   - Загружать спрайты только когда они нужны
   - Не загружать все декорации сразу

4. ОПТИМИЗАЦИЯ РЕНДЕРИНГА:
   - Кэшировать программно сгенерированные текстуры
   - Использовать offscreen canvas для сложных декораций

5. ПРОВЕРИТЬ ФАЙЛЫ:
   - Запустить скрипт check-missing-textures.js для проверки
   - Удалить или заменить отсутствующие файлы
