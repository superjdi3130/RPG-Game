# Решение проблемы с Live Share

## Проблема: Команда не работает, ссылка не появляется

## Возможные причины и решения:

### 1. Требуется авторизация в Live Share

Live Share требует авторизации через GitHub или Microsoft аккаунт.

**Решение:**
1. Нажмите `F1` или `Ctrl + Shift + P`
2. Введите: `Live Share: Sign In`
3. Выберите способ входа (GitHub или Microsoft)
4. Войдите через браузер
5. Попробуйте снова: `Live Share: Start Collaboration Session`

### 2. Расширение не активировано

**Решение:**
1. Нажмите `Ctrl + Shift + X` (открыть расширения)
2. Найдите "Live Share"
3. Убедитесь, что оно включено (Enable)
4. Если было отключено — включите и перезапустите Cursor

### 3. Перезапуск Cursor

Иногда расширение требует перезапуска редактора.

**Решение:**
1. Закройте Cursor полностью
2. Откройте Cursor снова
3. Попробуйте команду снова

### 4. Переустановка Live Share

Если ничего не помогает, переустановите расширение.

**Решение через терминал:**
```bash
# Удалить
code --uninstall-extension ms-vsliveshare.vsliveshare

# Установить заново
code --install-extension ms-vsliveshare.vsliveshare

# Перезапустить Cursor
```

### 5. Проверка через панель статуса

1. Посмотрите в правый нижний угол Cursor
2. Найдите иконку Live Share (обычно два кружка или "LS")
3. Кликните по ней
4. Попробуйте начать сессию оттуда

### 6. Проверка логов

Если проблема сохраняется, проверьте логи:

1. `F1` → `Live Share: Show Session Log`
2. Посмотрите, есть ли ошибки

### 7. Альтернативный способ запуска

Попробуйте через меню:

1. Откройте меню **View** (Вид)
2. Найдите **Command Palette** (Палитра команд)
3. Или используйте `Ctrl + Shift + P`
4. Введите `Live Share` и посмотрите все доступные команды

## Пошаговая диагностика:

1. ✅ Проверка установки: `code --list-extensions | findstr liveshare`
2. ⏳ Авторизация: `F1` → `Live Share: Sign In`
3. ⏳ Перезапуск Cursor
4. ⏳ Проверка команды: `F1` → `Live Share: Start Collaboration Session`

## Если ничего не помогает:

1. Удалите расширение полностью
2. Перезапустите Cursor
3. Установите заново через маркетплейс:
   - `Ctrl + Shift + X`
   - Поиск: "Live Share"
   - Установка от Microsoft
4. Перезапустите Cursor
5. Авторизуйтесь: `F1` → `Live Share: Sign In`
6. Попробуйте команду снова
